# "Smart" Alarm Clock

## Requirements
- Python 3.7
- virtualenv (`pip install virtualenv`)

## Setup

Initial create virtual env:
- `virtualenv --python=/usr/bin/python3.7 venv`
- `source venv/bin/activate`
- `pip install -r requirements.txt`

Run the app:
- `python app.py`
